# 研究室へのアクセス

- 研究室は、九州工業大学 飯塚キャンパス内にあります。
  [キャンパスへのアクセス](https://www.kyutech.ac.jp/information/map/iizuka.html)をご覧ください。
- キャンパスもより駅は、JR新飯塚駅です。<br>
  新飯塚駅からキャンパスまでは、スクールバスが利用できます。<br>
  [スクールバスの情報](https://www.iizuka.kyutech.ac.jp/campuslife/school-bus)をご覧ください。夏休み、年末年始などはスクールバスが減便されるのでご注意ください。

研究室は、下のキャンパスマップの 8番 総合研究棟 3階 N302 です。総合研究棟のエレベータを出て、すぐ前のドアをお入りださい。
